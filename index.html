Shape Royale Beta

Shape Royale is a game where you have to survive as long as you can in a battle royale. You can play as a square, circle or triangle.

Shape Royale is currently in beta and is not finished yet. It is currently only available on PC.

Shape Royale is made by Plazma Studios.

Shape Royale Update Log

Shape Royale Beta 1.0.0.0

This was the first update. It wasnt pretty, just a basic battle royale with no special effects. Each shape had sharp edges and pointy corners. Lots of bugs and just a hellhole, but that was what made it all begin!

Shape Royale Beta 1.0.0.1

This was the first update. It wasnt much different however, it did fix a lot of bugs like random crashes and unkillable enemys!

Shape Royale Beta 1.0.0.2

This was the second update. It was a very big one for the game as it came with trails, winnable games and many bug fixes. It also introduced the smooth shapes!

Shape Royale Beta 1.0.0.3

This update added a nice edition to the game, giving smooth gameplay, health upgrades and a bug free game!

Shape Royale Beta 1.0.0.4

This brings us the the current day! It is a beautifull update, bringing sound effects, cleaner trails, post processing and 3 new levels!
